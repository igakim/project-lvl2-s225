В рамках обучения на [Хекслете](https://ru.hexlet.io/), был реализован проект Вычислитель отличий.

Установка из npm:

```
npm i gendiff-igakim2
```

Сравнивает 2 файла и строит дифф. Поддерживаемые форматы - ini, yaml, yml, json

```
gendiff path/to/file.yml path/to/another/file.yml
```
